Welcome, to the batchData

This is an example of how to batchData and send when needed or possible.

Open you console and check the "network" and "application" to see how it is set data in localStorage and post to api/events when has 5 events.
(Can be configured in classes/batch.ts or when initiating new Batch());

Add data to batch { "eventName": "Event Type 1", "eventType": "clickEvent" }

Add different data to batch { "eventType": "Event Something Different", "somethingElse": "Something Else", "extraSubdata": { "subDataName": "My subDatadata", "subData": { "thisCanTake": "Anything" } }, "whatDidYouDo": "click" }
